== Description ==

WooCommerce EnvioPack te permite cotizar el valor de un envío desde la API de https://enviopack.com con una amplia cantidad de empresas de correo de una forma simple y estandarizada. 


== Installation ==

El plugin se mantiene actualizado para ser compatible con la última versión de Woocommerce y su instalación es bastante simple, consta de los siguientes pasos:

1) Instala y activa el plugin en tu tienda.

2) Ingresá a la opción “Ajustes” del menú Woocommerce, en el panel izquierdo de tu WordPress.

3) Acede a la pestaña “Envio”.

4) En el sub menú “Zonas de envío”, si no tienes una zona, agregar una zona de envío a preferencia.

5) Agregar dentro de la Zona de Envío, el metodo EnvioPack.

6) Completa y carga los datos necesarios. Las API las obtenes desde tu cuenta de https://enviopack.com, en la sección "Configuraciones de API".

7) Recorda, seleccionar y configurar direcciones, correos, tarifas, paquetes y reglas desde tu cuenta de https://enviopack.com.

 
Una vez configurado con tus datos, el plugin ya está listo para calcular los costos de envío en el carrito de tu tienda.

Tener en cuenta que los productos tienen que tener peso y dimensiones, caso contrario, no se mostrarán métodos de envío.

== Frequently Asked Questions ==

P. Desde dónde obtengo las API que me pide el plugin?.

R. Las obtenes desde tu cuenta de EnvioPack en la sección "Configuraciones de API". https://app.enviopack.com/configuraciones-api

P. Desde dónde genero mi dirección de origen?

R. La generas desde "Mi Perfil" en tu cuenta de EnvioPack. https://app.enviopack.com/configuracion/mis-direcciones
